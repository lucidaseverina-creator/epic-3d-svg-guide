## üîß KEY COMPONENTS

### 1. EffectsGallery Component

Main React component that:
- Manages effect selection and settings
- Renders the 3D scene
- Handles particle physics (for mesh-based effects)
- Provides UI controls for all effects

### 2. LavaParticle Class

Particle physics system for mesh-based lava:
- Position, velocity, acceleration
- Temperature-based coloring
- Gravity and cooling simulation
- Lifecycle management

### 3. ViscousLavaEngine Class

Physics engine for viscous lava:
- Inter-particle forces (viscosity, surface tension)
- Mouse interaction
- Particle lifecycle management
- Configurable physics parameters

### 4. EFFECTS_LIBRARY

Configuration object containing:
- Effect names and descriptions
- Shader settings for each effect
- Default parameter values
- Icon components

### 5. GLSL Shaders

Complete shader code for all 23 effects, including:
- Ray-marched metaballs (for glass effects)
- Mesh-based physics (for viscous effects)
- Various visual styles (neon, fire, ocean, etc.)

## üîç TECHNICAL DETAILS

### Ray-Marching
- Uses SDF (Signed Distance Function) for metaballs
- 96 raymarch steps per pixel
- Multiple blob support (up to 24)
- Configurable radius, speed, and shininess

### Mesh-Based Physics
- Particle-based simulation
- Inter-particle forces (viscosity, surface tension)
- Temperature-based coloring
- Gravity and cooling simulation

### Shader Features
- Fresnel reflections
- Specular highlights
- Color gradients
- Noise-based patterns
- Temporal animation
